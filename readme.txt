Adds a new button into the visual editor within Wordpress, allowing easy insertion of the HuStream Interactive Player into pages and posts.

== Description ==

This plugin enables a new short-tag in the visual editor within wordpress that takes care of embedding the HuStream Interactive Player into any page or blog post, along with a toolbar item to automate the creation of these tags.

== Installation ==

1. Unzip `hustream-social-video.zip` into your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit or create a page/post and you should see a new HuStream icon on your toolbar.

== Frequently asked questions ==

= Where do I get Video URLs from? =

Once you create an interactive video using the HuDesigner (http://clients.hustream.com/user/signup), you click publish and the URL will be presented to you under the Settings & Publishing dialog after it is ready.
